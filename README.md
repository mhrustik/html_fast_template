Gulp 4 Html Optimize (scss version)
=====================

### Install:

    npm i
    npm i --global gulp-cli
    gulp watch
    gulp build

Folder          | Description
----------------|----------------------
app             | Dev folder (command: gulp watch)
build           | Prod folder (command: gulp build)

### Included css libraries:

    normalize
    flex-grid
    fontawesome5
